## 📌 Overview
ShoppingMart is a **pure HTML & CSS** based website showcasing multiple shopping categories such as:
- 👟 Shoes  
- 🏬 Big Bazar products  
- 🪑 Urban Ladder furniture  
- 🏠 Home essentials  
- 🍎 Fruits & groceries  

No JavaScript is used, making it **lightweight, fast, and easy to maintain**.
